1. Game Overview: "GATE"
"GATE" is a first-person puzzle-platformer where the player must navigate a treacherous testing facility. The core mechanic is a teleportation tool that allows the player to instantly warp to specific valid surfaces.
Objective: the player must complete puzzles to advance to the end
2. How to Play (Controls)
W, A, S, D: Move forward, left, backward, right.
Spacebar: Jump.
Left Mouse Click: Fire Teleportation Laser.
Note: You can only teleport to walls tagged as "TeleportWall". You cannot fire the laser again until you have touched the ground to reset the mechanism.
Mouse Movement: Look around.
